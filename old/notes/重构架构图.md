## 目标架构（Java + Python LangChain）

```
                         Natural Language Query
                                 │
                    ┌────────────┴────────────┐
                    │                         │
            ┌───────▼───────┐          ┌──────▼───────┐
            │  Java 确定性   │          │   Python    │
            │    逻辑层       │◄────────►│  LangChain   │
            │               │  gRPC/   │     层       │
            │ • SQL 编译     │  REST    │              │
            │ • 权限控制     │          │ • Memory     │
            │ • SQL 执行     │          │ • Intent     │
            │ • 结果格式化   │          │   Parser     │
            │               │          │ • RAG        │
            │               │          │ • Agent      │
            └───────────────┘          └──────┬───────┘
                                                │
                                    ┌───────────┴───────────┐
                                    │                       │
                            ┌───────▼───────┐       ┌──────▼───────┐
                            │  Conversation  │       │   Vector DB   │
                            │    Memory       │       │  (FAISS)      │
                            │  (历史上下文)   │       │ (指标定义)    │
                            └───────────────┘       └──────────────┘
```
